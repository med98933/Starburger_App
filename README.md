## Starburger_app


## Built With

* [Node.Js](https://nodejs.org)
* [MongoDB](https://www.mongodb.com)
* [Tailwind](https://tailwindcss.com)
## Installation

Clone the repo:

```bash
git clone https://github.com/med98933/Starburger_App.git
```

### Usage

```bash
cd client
npm install
npm start
```

In other terminal

```bash
cd server
npm install
npm start
```
